# Swiggy-Sales-Analytics-Dashboard#

## 📌 Project Overview

This project analyzes Swiggy food delivery data to understand customer behavior, restaurant performance, and ordering patterns.
The analysis helps identify trends that can improve business decisions for food delivery platforms.

---

## 🎯 Objectives

* Analyze food delivery order data
* Identify popular restaurants and cuisines
* Understand customer ordering patterns
* Extract insights from the dataset using Excel

---

## 📂 Dataset

The dataset contains information about:

* Restaurant Name
* Location
* Cuisine Type
* Rating
* Cost for Two
* Delivery Time
* Customer Reviews

---

## 🛠 Tools Used

* Microsoft Excel
* Data Cleaning
* Data Analysis
* Pivot Tables
* Data Visualization

---

## 📊 Key Analysis Performed

* Restaurant rating analysis
* Cost analysis for different restaurants
* Popular cuisine identification
* Location-based restaurant distribution
* Customer review insights

---

## 📈 Insights

* Identified top-rated restaurants based on customer reviews
* Found cost trends for different cuisine categories
* Highlighted areas with the highest restaurant density
* Observed patterns in delivery time and ratings
